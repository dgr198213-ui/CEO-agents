# Contribuir a CEO-Agents 🚀

¡Gracias por tu interés en contribuir a CEO-Agents! Este es un proyecto Open Source y damos la bienvenida a todas las contribuciones, desde correcciones de errores hasta nuevas funcionalidades.

## ¿Cómo puedo contribuir?

### Reportar Errores
Si encuentras un error, por favor abre un **Issue** en GitHub detallando:
- El comportamiento esperado y el real.
- Pasos para reproducir el error.
- Tu entorno (versión de Nim, OS, etc.).

### Proponer Funcionalidades
Si tienes una idea para mejorar el sistema:
1. Abre un **Issue** para discutir la idea.
2. Si se aprueba, puedes proceder con un Pull Request.

### Enviar Pull Requests (PRs)
1. Haz un **Fork** del repositorio.
2. Crea una rama para tu cambio: `git checkout -b feat/nueva-funcionalidad`.
3. Asegúrate de que los tests pasen: `./scripts/run_tests.sh`.
4. Haz commit de tus cambios siguiendo [Conventional Commits](https://www.conventionalcommits.org/).
5. Envía tu PR a la rama `main`.

## Guía de Estilo

### Backend (Nim)
- Sigue las convenciones oficiales de [Nim Style Guide](https://nim-lang.org/docs/nep1.html).
- Documenta las funciones públicas con comentarios `##`.
- Mantén las funciones pequeñas y con una única responsabilidad.

### Frontend (Next.js)
- Usa TypeScript para todo el código nuevo.
- Usa componentes funcionales de React y hooks.
- Estiliza con TailwindCSS siguiendo el diseño existente.

## Comunidad y Código de Conducta
Por favor, mantén un ambiente profesional y respetuoso. Valoramos la diversidad de ideas y la colaboración constructiva.
